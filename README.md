# 🛡️ SOC-Lab-Investigations
**Analyst:** Derrick Ernest Perez | **Focus:** Incident Triage & Threat Hunting

### 🎯 Goal
Bridging theory and practice by documenting hands-on investigations into real-world security alerts to build professional SOC workflows.

### ⚙️ Investigation Workflow
Every case study follows the standard Incident Response lifecycle:
1. **Analyze:** Triage the alert and establish context.
2. **Pivot:** Correlate logs across the network to reconstruct attacker activity.
3. **Conclude:** Classify as True or False Positive.
4. **Remediate:** Document defensive actions and follow-up.

### 🛠️ Toolkit
* **Platforms:** Splunk, ELK, TryHackMe
* **Techniques:** Log Correlation, Phishing Triage, Network Traffic Analysis
* **Operational Security:** All indicators (IPs/URLs) are professionally **defanged** (e.g., `hxxp://example[.]com`).
